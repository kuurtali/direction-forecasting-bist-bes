## D.2 Formuller
Precision           = TP / (TP + FP)
Recall (Sensitivity)= TP / (TP + FN)
F1                  = 2 x (Precision x Recall) / (Precision + Recall)
Specificity         = TN / (TN + FP)
Balanced Accuracy   = (Sensitivity + Specificity) / 2

POOLED ACCURACY     = (TP_seed1 + TP_seed2 + TP_seed3 + TN_seed1 + TN_seed2 + TN_seed3) / N_pool
                    = (sum_TP + sum_TN) / (3 x N_test)

3-SEED MEAN ACCURACY = (Acc_seed1 + Acc_seed2 + Acc_seed3) / 3
                     = ortalama; matrisin TP/TN sayilariyla birebir ortusmek
                       zorunda degildir (seed varyansini hosgorur).
