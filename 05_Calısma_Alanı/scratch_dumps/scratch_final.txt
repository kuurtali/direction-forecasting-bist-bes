Theoretical Framework â€” ARIMA
AutoRegressive Integrated Moving Average as parametric baseline
Model form. 
An ARIMA(p, d, q) model specifies that the d-th difference of the observed series can be expressed as a linear combination of p past values and q past innovation terms. Applied to weekly BES fund log-returns, ARIMA provides a parametric benchmark against which the deep-learning models are compared.
Hyperparameter strategy. 
Following Van der Burgt (2023), autoregressive and moving-average orders are tied to the input-window length: p = q = input_len âˆˆ {2, 4, 6}. The integration order d is grid-searched over {0, 1, 2, 3, 4, 5} on the validation set. The best d per window is retained and the final model is refitted on train+val before test evaluation.
Fitting. 
Conditional Sum of Squares (CSS) is used in place of the default Maximum-Likelihood estimator. CSS is numerically more stable on short TEFAS samples where the innovation variance estimate can otherwise fail to converge (Hamilton, 1994).
Directional classification. 
Point-forecasts are mapped to a binary Up / Down decision by comparing the forecast to the last observed price; accuracy is then computed on this binary outcome.

A broad MC definition as first-class safeguard against the accuracy illusion
What is Majority-Class collapse? 
A binary classifier collapses to the Majority Class when it outputs the same label for (almost) every test observation. On imbalanced data this can produce high raw accuracy while the model has learnt nothing â€” the Accuracy Illusion (Valverde-Albacete & Pelaez-Moreno, 2014).
Broad MC definition used here. 
A configuration is flagged MC if ANY of the following holds:
  â€¢ Spec = 0  â€” no Down moves predicted
  â€¢ Sens = 0  â€” no Up moves predicted
  â€¢ Spec = NA â€” test set has zero Down observations
Why broad, not narrow? 
A narrow definition (Sens = 1 AND Spec = 0) would miss configurations that predict, e.g., 35 out of 37 observations as Up â€” they clearly exhibit MC behaviour but would slip through. The broad definition eliminates that loophole.

â€¢ Valverde-Albacete & Pelaez-Moreno (2014) â€”

â€¢ KaradaÄŸ Erdemir & KÄ±rkaÄŸaÃ§ (2024) â€”
