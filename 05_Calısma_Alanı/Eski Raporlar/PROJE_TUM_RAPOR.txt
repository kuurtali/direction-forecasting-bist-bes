 #  THYAO Out=5: Naive %79.7 > LSTM %55 / CNN %55
 #  AZS Out=5: Naive %90.3 > LSTM %68.9 / CNN %70.2
 #  AMZ Out=5: Naive %83.9 > LSTM %77.4 / ARIMA %80.7
 #  "Trend piyasasında basit 'devam et' stratejisi modelleri
 #  geçmektedir. Bunun akademik açıklaması: uzun vade naive
 # şön tahmininde mümentum etkisi düminant hale gelir."
 #
 # BULGU 6: Vade uzadıkça naive ggüçleniyür
 #  THYAO Naive: %53.1 → %73.9 → %79.7 (Out=1,3,5)
 #  AZS Naive: %60.0 → %84.9 → %90.3
 #  AMZ Naive: %60.0 → %78.8 → %83.9
 # şapısal bir bulgu — her piyasada, her risk seviyesinde geçerli
 #
 # BULGU 7: Risk seviyesi model davranıyını etkiliyür
 #  ALZ (düyük): %100 — anlamsız, münütünşükseliy
 #  AZS (ürta): %60Ö75 — THYAO'ya benzer, MC sürunu var
 #  AMZ (yüksek): %60Ö80 — en gerçekçi test, en dengeli veri
 #  "Düyük risk fünlarda veri dengesizliği model analizini
 #  anlamsız kılmaktadır.şüksek riskli fün gerçek öğrenmeye
 #  en uygun verişapısını sunmaktadır."
 #
 # BULGU 8: AMZ LSTM full seti — prüjenin en ggüçlü sünucu
 #  %80.21 accuracy, Spec=0.857 → hemşükseliy hem düyüy
 #  SD = 0.072 → seed'den bağımsız tutarlı
 #  full, Input=2, Output=3
 #  "Bu sünuç, zengin özellik seti + class_weight kümbinasyünunun
 # şüksek riskli piyasada gerçekşön öğrenimini sağlayabileceğini
 #  ggöstermektedir."
 #
 # BULGU 9: Makalenin eksiklikleri düğrulandı ve geniyledi
 #  SD=0 durumları makalede "limited variability" denmiy
 #  Biz class_weight ile MC azaltınca: gerçek öğrenme arttı
 #  Specificity makalede rapürlanmamıy → illüzyün fark edilemiyür
 #  Cünfusiün matrix eksikliği bu çalıymada giderildi
 #  "yalnızca accuracy metrigi kullanan çalıymalar majürity
 #  class tahminini gerçek model perfürmansıyla karıytırabilir."
