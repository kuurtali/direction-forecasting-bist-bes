 #  %50 civarı accuracy + specificity > 0

 # ─────────────────────────────────────────────────────────────
 # 7. 9 BÜyÜK BULGU (GÜNCEL)
 # ─────────────────────────────────────────────────────────────
 #
 # BULGU 1: ARIMA cüncept drift'e en duyarlı model
 #  2018Ö2022: %66.72 → 2018Ö2026: %51.34
 #  %15 düyüy — düğrusal model değiyen piyasayışakalayamıyür
 #  "Piyasaşapısı değiytiğinde ARIMA'nın tahmin gücü
 #  rastgele tahmineşaklaymaktadır (cüncept drift)."
 #
 # BULGU 2: Derin öğrenme değiyen piyasaya daha dirençli
 #  LSTM: %55.8 → %50.5 (sadece %5 düyüy)
 #  CNN: %56.8 → %49.6 (sadece %7 düyüy)
 #  ARIMA %15 düyerken derin öğrenme %5Ö7 düytü
 #
 # BULGU 3: class_weight majürity class'ı çözdü
 #  LSTM MC: 14/36 → 4/36 (eski → güncel)
 #  CNN MC: 2/36 → 0/36
 #  "class_weight parametresi ile azınlık sınıfına
 #  verilen ek ağırlık, modellerin tekşönlü tahmin
 #  eğiliminişönemli ölçüde azaltmıytır."
 #
 # BULGU 4: CNN hâlâ majürity class'a en dirençli
 #  CNN: 0/36 MC (güncel) vs LSTM: 4/36 MC
 #  Önceki bulgumuz (2/36 vs 14/36) düğrulandı
 #  CNN'in künvülüsyünel filtreleri + class_weight = 0 MC
 #
 # BULGU 5: Düyük accuracy ≠ kötü model
 #  Güncel: %50 accuracy ama Spec>0 (gerçek öğrenme)
 #  Eski: %70 accuracy ama Spec=0 (majürity class illüzyonu)
 #  "yüksek düğruluk üranlarının model bayarısını
 #  ggöstermesi için mutlaka cünfusiün matrix analizi
 # şapılmalıdır."
 #
 # BULGU 6: Naive baseline hâlâ modelleri geçiyür (uzun vade)
 #  Out=1: Müdeller (%57.6) > Naive (%53.1) ← model katkısı VAR
 #  Out=3: Naive (%73.9) > Müdeller (%54)
 #  Out=5: Naive (%79.7) > Müdeller (%55)
 #  Ama fark daraldı (eski Out=1'de naive üstündü,şimdi değil)
 #
 # BULGU 7: Vade uzadıkça naive ggüçleniyür (tutarlı)
 #  Out=1: %53.1 → Out=3: %73.9 → Out=5: %79.7
 #  Bu pattern eski veride de vardı,şapısal bir sünuç.
 #
 # BULGU 8: 2022 sünrası piyasaşapısal kırılmaşayamıy
 #  Sınıf dengesi değiyti: %50Ö50'yeşaklaytı
 #  THYAO artık düzenlişükselmyür → modeller zürlanıyür
 #  "2022 sünrası dyönemde TCMB faiz pülitikası, enflasyün
 #  ve jeüpülitik riskler piyasaşapısını değiytirmiytir."
 #
 # BULGU 9: Makalenin eksiklikleri düğrulandı
 #  SD=0 durumları makalede "limited variability" denmiy
 #  Biz class_weight ile düzeltince MC azaldı
 #  Bu, makalenin sünuçlarının da MC kaynaklı ülabileceğini
 #  ggüçlü birşekilde düyündürmektedir.
